**Complete MERN Stack Education Platform with AI Integration**

A comprehensive education management system that connects teachers and students with AI-powered features for question generation, assessment, and performance tracking.

## 🛠️ **Tech Stack**

**Backend:**
- Node.js + Express.js
- MongoDB + Mongoose (8 comprehensive schemas)
- JWT Authentication + Role-based access
- Socket.io for real-time features
- Multer for file uploads
- Email notifications (Nodemailer)
- Security middleware (Helmet, Rate limiting)

**Frontend (Ready for Development):**
- React.js with modern hooks
- State management ready
- API integration services prepared
- Responsive design foundation

**AI Integration:**
- Multi-tier AI provider support (Free → Premium)
- Question generation system
- Automated grading capabilities
- Smart recommendation algorithms

## 🚀 **Quick Start Deployment**

### **1. Backend Deployment (Railway):**
```bash
git clone https://github.com/DhimanTarafdar/gyanguru_education_platform.git
# Follow RAILWAY_DEPLOYMENT_STEP_BY_STEP.md
```

### **2. Database Setup (MongoDB Atlas):**
- Free 512MB cluster
- Auto-configured schemas
- Production-ready security

### **3. Environment Configuration:**
- Copy `.env.example` to `.env`
- Configure MongoDB URI
- Setup email service
- Add JWT secrets
